# General Settings — Settings

> **Module:** Website · **Screen:** General Settings · **Route:** `/website/settings` · **Status:** Draft

## Purpose
Configure core website settings — site identity, analytics, social, and scripts.

## Layout
Tab-based settings page (no Sheet drawer).

## Tabs

### Tab 1: General
- Site Name
- Site Description (tagline)
- Site Language (dropdown — Core Localisation)
- Timezone
- Date Format
- Copyright Text

### Tab 2: Theme & Branding
- Primary Color (color picker)
- Secondary Color (color picker)
- Heading Font (Google Font selector)
- Body Font (Google Font selector)
- Logo (media picker — recommended SVG/PNG)
- Favicon (media picker — 32×32px)
- Custom CSS (code editor)

### Tab 3: Analytics & Scripts
- Google Analytics ID (GA4)
- Google Tag Manager ID
- Facebook Pixel ID
- Custom Head Scripts (textarea)
- Custom Body Scripts (textarea)

### Tab 4: Social Media
- Facebook URL
- Instagram URL
- Twitter/X URL
- LinkedIn URL
- YouTube URL
- WhatsApp number

### Tab 5: Maintenance
- Maintenance Mode (toggle + message editor)
- Coming Soon Mode (toggle)
- Password Protection (toggle + password field)

## Actions
- **Save** (per tab)
- **Preview site** (opens storefront in new tab)
