# Post List — Blog

> **Module:** Website · **Screen:** Blog Posts · **Route:** `/website/blog/posts` · **Status:** Draft

## Purpose
Manage all blog posts — list, create, edit, publish.

## Layout
Standard list page + right Sheet drawer for metadata. Content editing in full-page rich text editor.

## Table Columns
`Title` · `Category` · `Author` · `Status` · `Published At` · `Views` · `Actions`

## Filters
Status · Category · Author · Date range

## Sheet — Post Form Fields
- Title (required)
- Slug (auto-generated)
- Excerpt
- Thumbnail (media picker)
- Category (dropdown)
- Author (user picker)
- Tags (multi-select)
- Status (draft / scheduled / published)
- Scheduled Publish Date (if scheduled)
- SEO Meta Title / Description

## Actions
- **Edit Content** → opens rich text editor (full page)
- **Publish** / **Schedule**
- **Duplicate**
- **Translate** (AI tool)
- **Delete**
